# Project IGI Web
A browser-based, 3D tactical stealth FPS game built with Babylon.js and Vite.

## Features
- First-Person Shooter controls
- Cross-platform support (PC and Mobile)
- Havok Physics Engine integration
- Entity-Component-System architecture (ECS)
- Dynamic Enemy AI

## Setup
1. Clone the repository.
2. Run `npm install`
3. Run `npm run dev` to start the local development server.
